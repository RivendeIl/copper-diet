# 윌슨병 구리 식품 가이드

윌슨병 환자를 위한 식품별 구리 함유량 검색 도구와 분류표입니다.

## 바로 사용하기

- **[식품 검색](https://rivendeil.github.io/copper-diet/)** — 식품명을 입력하면 구리 등급을 바로 확인
- **[인쇄용 분류표](https://rivendeil.github.io/copper-diet/print.html)** — A4 출력용 (Ctrl+P / Cmd+P)

> iPhone에서 Safari로 접속 → 공유 → "홈 화면에 추가"하면 앱처럼 사용할 수 있습니다.

## 등급 기준 (100g당)

| 등급 | 기준 | 지침 |
|------|------|------|
| 🔴 초고함량 | ≥ 1.0 mg | 절대 금지 |
| 🟠 고함량 | 0.3 ~ 1.0 mg | 피할 것 |
| 🟡 주의 | 0.2 ~ 0.3 mg | 제한적 섭취 |
| 🟢 저함량 | < 0.2 mg | 자유 섭취 |

- 근거: 세브란스병원 영양상담, CRMR Wilson (프랑스), Wilson Disease Association
- 1일 구리 섭취 목표: **1mg 이하**
- 같은 식품이 조리법에 따라 달라질 수 있어 **최대 함유량** 기준으로 분류
